# 🥋 Jones Kenpo Student App — Feedback

![Jones Kenpo Student App Logo](images/jones-kenpo-logo-text-left-and-right.png)

Welcome!  
This repository is a place for **Jones Kenpo students** to share feedback, report bugs, and suggest ideas for the Student App.

The app is designed to help you:
- track your progress  
- watch videos of techniques, sets, and forms  
- stay connected to your training  

Your feedback helps make the app better for everyone.

---

## 💬 How to Share Feedback

You can send feedback **any way you like**:
- GitHub Issues (preferred for organization)  
- WhatsApp  
- Email  
- TestFlight’s built‑in feedback  
- Or just tell me in person  

If you’re comfortable using GitHub, it’s the easiest way for me to track things.

👉 **To submit feedback here, click “Issues” → “New Issue”**  
You’ll see simple forms for:
- Bug reports  
- Feature ideas  

---

## 🔒 A Quick Note About Privacy

This feedback forum is **public**, which means:
- Anyone can read the issues  
- Please **don’t include personal information**  
- Avoid sharing private student details or anything sensitive  

If you need to share something private, use WhatsApp or email instead.

---

## 🏷️ What Happens After You Submit

I’ll review each submission and:
- ask questions if needed  
- mark it as “in progress” when I’m working on it  
- mark it as “resolved” when it’s fixed or added  

You’ll get updates automatically if you have a GitHub account.

---

## 🙏 Thanks for Helping

Your feedback directly shapes the app.  
Thanks for taking the time to help improve the training experience for all students.
